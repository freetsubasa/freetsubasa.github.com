What's in my computer

寫在前面

在看了 m157q 大大的我平常在電腦上用了哪些程式以及 資安網紅 DuckLL 大大寫的 我的電腦環境後，一直很想要自己寫一篇類似的東西，也剛好前陣子狄卡還滿流行 what's in my ooo 之類的標題，就拿來用一下囉 XD

最近也一直在重新建環境，就來開一篇來記錄一下自己比較常用到的是那些工具，之後如果有發現新的更好用的會再 update 上來吧
如果我記得的話

OS & VM & VPS

  • Windows 7 / Windows 10
    • 毫無反應，只是拿來打遊戲用的（誤）
    • 最近的 Windows 10 其實還滿好用的，介面的流暢度還不錯，所需的資源也不是說到非常多
    • Office 系列也是我遲遲無法在工作環境捨棄掉 Windows 的原因之一
    • 不過在休閒方面真的完全沒考慮過要捨棄 Windows 就是了
  • Ubuntu
    • 最新的 LTS 版本正好出了
    • 閒閒沒事打打 CTF 的時候會用到
      • 以前是用 Kali Linux 整套包好的，後來覺得太肥就還是改用 Ubuntu ，要什麼套件再自己裝
    • 明明覺得 Ubuntu 在相依性的部分還是略雷，但不知道為什麼開發環境還是很愛用他
  • CentOS
    • 目前手邊有在管理的機器基本上都是 CentOS
    • 學校兩大愛用款之一，另一個是 FreeBSD
    • 真的穩
    • 而且我跑了至少學校還比較好找人繼續管 QQ
  • Virtual Box
    • 做測試跟開發需要的環境基本上都裝在這了
    • 有些東西還是懶得上（或是不太能上） VPS　基本上就會丟到這測試
    • 網路設定一直是個常常被 google 的問題，之前留下的筆記也常常被我自己 google 到 XDDD
  • Google Compute Engine
    • 因為有免費提供的 300 鎂試用，所以一直掛在這上面
    • 有台灣機房，用起來還滿快的，ssh 過去不會有延遲的感覺
    • 之前有拿來架給 HITCON GIRLS 的 CTF 網站在這上面
    • 目前使用上沒什麼問題，文件也還算豐富，雖然有時候會遇到沒有中文翻譯的文件就是了

Editors

  • Vim
    • 真的是用習慣了，連寫純 html 都可以用 vim 寫得很開心
    • 但我還是習慣用方向鍵來移動（廢）
    • 搭配 Vundle 來裝 plugin 還滿方便的，可以玩一個下午沒問題
    • 目前最低基本需求這些套件
      • youcompleteme
      • vim-snipmate
  • Sublime Text
    • 其實我只是單純拿來當作記事本在用，聽說有很多有趣的 plugin 可以玩看看
    • 偶爾拿來隨手記錄一些筆記，畢竟整個關掉後再打開會回到原本的頁面，拿來當作臨時便條紙的感覺

Terminal

  • iTerm2
    • 說真的離開 Mac OS X 的環境之後真的很難找到差不多好用的東西了
    • 不用在意畫面重繪問題或是寬度問題真的很棒 QQ
  • Pietty
    • 轉換到 Windows 環境後試了不少東西，最後還是老面孔的 pietty 最安定了
    • 至少我不用煩惱快捷鍵互衝的問題
    • 如果沒什麼複雜的需求的話我覺得 pietty 的夠用了 QQ
  • Fish Shell
    • 這 shell 真的超有趣的，Autosuggestions 這功能真的好用
    • Web Base 的設定方式也很好玩
    • 如果在 Mac OS X 上面想試試看除了 bash、zsh 以外的 shell 的話可以看看這個
  • Bash
    • 基本上就是在寫 shell script 時候會用到
    • 各種因緣際會下翻過使用手冊，有很多神奇的寫法
    • Advanced Bash-Scripting Guide

CLI Tools

  • screen & tmux
    • 早些時候一直是 screen 派的，但後來在切割畫面之後發現 tmux 還是比 screen 直覺點
    • 不過目前還是兩個都用就是了，不過之後應該會漸漸以 tmux 為主就是了
    • 第一次接觸 screen 這東西是因為有人跟我說用 mosh + screen 你就可以在研討會的網路下生存的很好 XD
  • htop
    • 資源監控
    • 有顏色跟精美排版看起來心情就是會比較好
    • 理論上會是我 screen 第一個分頁，但是我常常把不小心把它關掉就是了
  • mtr
    • 查看網路用的，強化版的 traceroute
  • misc
    • 比較常用的還有一些假日在 CTF 打雜的時候會用到的工具
    • 種類比較多也很雜，就改天再說吧

Online Service

  • Google 全家餐
    • 包含 gmail、google calendar、google map（iOS 內建的地圖真的是爛的可以）、google drive
    • Calendar 只要在標題註明時間地點的那個功能真心好用 T^T
  • Trello
    • 一開始是為了打 CTF 才註冊的，後來就把他當成筆記在使用 XD
    • 出國規劃行程用這個也不錯，有離線瀏覽功能可以把一些緊急事項放上去，就算網路中斷也可以有種安心感 (?)
  • Slack
    • 多人協作聊天室，但說真的不會用的人就是不會用 XD
    • 一個 workspace 一個帳號密碼的設計略擾人（還好有 magic link …）
  • Discord
    • 第一次接觸的感想就是 RC Style (?)
    • 結果只拿來掛某兩個群組 XDDD
  • Steam
    • 我的錢包的化身
    • 最近的樂趣是玩獨立小遊戲
  • HackMD
    • 線上目前用過最好用的 markdown 編輯器
    • 最近的功能越來越好用了
    • 他們的 github
  • Social Network
    • Facebook、Twitter、Plurk、Instagram、Line、Telegram
    • 這年頭就算是個邊緣肥宅，也可以裝超多社群軟體ㄛ
