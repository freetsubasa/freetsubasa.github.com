寫在前面

好像又是個充滿驚喜的一年，大概啦
今年過的速度感覺比起以往都快上許多，大概是因為有在工作的關係吧 …

決定這次的年度回顧文放的是張懸的這首歌，很喜歡這句歌詞
希望有機會可以去聽現場，不過按照我說說的個性，估計也只是說說罷了 XD

我擁有的都是僥倖啊
我失去的都是人生

關於 sudo

sudo 又胖了，跟我一樣穩定橫向發展中
真不愧是跟我很像的小笨貓

2019 年就要滿四歲了，每次幫 sudo 過生日就像是某種倒數計時一樣
好貪心的希望你可以再陪我更久更久一點
好幾次都在思考自己是不是應該尋找一下信仰
這麼怕生的你不知道能不能在黃泉路上安穩走完

喜歡你每天早上跟我擠棉被
喜歡你每天晚上跟我擠沙發
喜歡你蹭我的手
喜歡你軟軟的對著我叫
好喜歡好喜歡你，這世界上我最喜歡的就是你了

剛帶 sudo 回去的時候隔壁的鄰居說

有個溫暖的毛孩在家裡等你回家的感覺真好

真的，我也這麼覺得
有你在的地方，對我來說都是家

希望新的一年可以帶你出去曬曬太陽，踩踩草皮
搬到一個可以讓你繼續看車車跟鳥鳥的地方

小笨貓，我真的好愛你

關於 Security

放飛自我的一年 XD

工作之後真的很少碰 CTF 了（雖然本來就不多了，ㄏㄏ），倒是有跟 mike 重新連繫上
年底有一起打了幾場

人生第一次參與 bounty，也算是某種程度上的成就解鎖了吧
覺得很有趣就是了
如果可以的話還是希望 2019 可以繼續走在這條路上

真．跟 2018 年初的我講自己都不會相信的事

噗浪跟風，但是跟了之後一直忘記補完，就留到這邊寫完吧
老樣子，年初的事情久遠的跟兩年前一樣

  • 你真的可以畢業了，雖然你畢業之後還是常常被學校抓去幫忙修機器
    • 鬆餅屋換老闆了，紅茶也沒有以前好喝了
    • 餐車早就消失了記憶的深處了
    • 樓下那間越南河粉外帶好吃多了
  • 你會搬到一個沒有車就沒有腳的地方
    • 然後 HITCON 的時候會被當倉庫使用
    • 然後連鄰居都不一定會講上話
    • 找不到什麼好吃的
    • 麥當勞外送不會到的地點
    • 速食店外送肯來的只有 pizza
    • 哦哦對，因為算是新北，所以你只有 Uber Eats 喔
  • 你找的到工作耶，棒棒
    • 你鼓起勇氣投的兩間公司，一間你上了，另一間拖了很久才打電話問你面試 XD
  • 你會到美國領地上旅遊
    • 還會跑去打靶跟潛水
  • 你會跟某人漸行漸遠
    • 你以為你可以忍的其實你不能
    • 你以為你可以接受的其實你不能
  • 你會活得好好的

關於自己

心理狀態一直都很差
躁期跟鬱期之間的表現越來越明顯
覺得對身邊的人很抱歉，但是自己卻很難很難控制自己

於是選擇逃避，於是選擇離開
活著的每一天都是煎熬的感覺其實真的滿痛苦的
如果沒有 sudo 在，我還真的不知道自己還能撐多久

然後我好胖，我好廢 QQ

關於你

不知道這樣對不對
不知道掙扎的結果會怎麼樣

好想逃，但不能逃

不想一直被誤會，不想一直解釋根本不存在的東西
不想被強迫，好想逃

雜談 & 2019 年的願望

  • 希望可以好好活著
  • 希望可以變得更強

希望可以跟 sudo 繼續快快樂樂的生活在這莫名其妙的世界
