COSCUP 2015 一點心得

2015-08-17
Conf

Topic List

  1. 寫在前面
  2. Day 0
  3. Day1
  4. Day2
  5. After COSCUP

寫在前面

剛好在 twitter 上面看到有人貼線路組招募資訊，
小飛機也是一臉「我想教你但是我沒空，你就自己去學吧」的臉 (?)

所以我就傻傻的去報了線路組，其實我本來以為我會是砲灰的 orz

結果我還真的收到錄取通知，
收到的那一刻我都快傻啦，真的出乎我的意料之外 QQ

Day 0

早上莫名的在交流道塞車，結果變成到中研院的時間剛剛好 XD

然後就先去 R1 放東西，
在滑手機的時候應該是場務組的人跑來跟我聊天，
他就問我說線路組要幹嘛，還有一些奇奇怪怪的問題XD

然後就跟我要 FB 了 XDDDD

中午吃飯的時候感謝 xxx 跟 JohnDoe 跑來跟我搭話，
不然我覺得我會尷尬到整個活動結束 QQ

下午就開始去場佈拉線放 AP 有的沒的，
拉的線如果被小飛機看到一定會被嫌棄XDDDDD

大概就是會嫌拉的不夠直，膠帶貼的不夠漂亮之類的吧，我猜

反正等小飛機第二天進到 H2 辦 SDN Workshop 的時候我人也不會在現場 (大誤)

然後晚餐吃摩斯，
吃完晚餐整完最後的線路之後就跟其他線路組組員坐在地上聊天

原來 SDNDS.TW 的成員也有在線路組的，
也有今年剛升上大一的，也有國中生，高中生，不知道為什麼突然覺得自己好老 QQ

晚上跟我一起睡的是一個人很好的姊姊 >w<

然後我 11 點以前就陣亡了，顯然是因為最近生活過的太規律 XDD

無法晚睡的日子好痛苦啊

Day1

早上把攤位的網路重新測了一次，但是 2.4G 的網路還是不穩得很嚴重 QQ

只能請廠商插有線網路，剛剛好有場沒帶轉接頭跟網路線，
於是我就把我身上的轉接頭跟網路線借出去了，囧a

然後有拿到 MySQL 可愛的海豚，攤位上面的人好好哦 QQ

下午晃著晃著小飛機就出現了，來自四面八方的各種通報 (?)

順便跟小飛機拿了 Thunderbolt 轉 RJ45 嶄新的閃亮亮的 <3

晚上 Bof 去了線路組的，
除了有 pizza 還有好吃的烤鴨耶 \0w0/

後半段時間路過一下 CTF 分享會 (是貨真價實的路過)，
人還滿多的，一群黑黑

後面就是中部地區大食團，
他們的 pizza 多的莫名，我這麼晚去竟然還有剩下的 pizza 沒有吃完 XD

然後終於看見傳說中的 jc 長什麼樣子，
因為那天晚上有超多人問我有沒有看到 jc 在哪裡 XDD

Day2

早上超級早起的去拍照，空拍機超帥氣的啊啊啊啊啊啊啊

不知道可以操控的最高跟最遠距離是多少，因為整台看起來真的滿大台的

中午幫忙小飛機搶 Lighting Talk ，一個搶到順位一，覺得我自己有點厲害

下午就幫忙徹場，搬東西，然後去聽期待的 LightingTalk XD

  • Bind Python and C by StarNight
    • 這個講者講的內容我覺得值得一個放入議程裡面
  • 後山大開源計畫
    • 東部學生資訊社團熱招生中 XD
  • 三分鐘教阿嬤學會打中文
    • 主題很有趣，擺攤的時候被攤位的裝飾有點嚇到，一時之間沒有把攤位跟這場 LT 連在一起
  • 傳統產業的開源殉道者 by derek
    • 聽了我都想哭了，在老闆眼中開源軟體 = 免費 QQ
  • TIH打一下廣告～
    • 對對哥
  • 雲端巨量資料物聯網深度學習文化創意產業之前進吧！高捷少女
    • 用說的有點難，直接上影片最快XD

After COSCUP

首先是要謝謝線路組的各位前輩們，對我真的是十分的照顧，我在這三天雖然有點累但是很開心 :D

能加入線路組真的是一件很棒很棒的事情

今年的 COSCUP 很棒，很好玩，謝謝辛苦的工作人員們！
